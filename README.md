## рҹҢұ What is CropCare AI?
**CropCare AI** is an advanced, full-stack AI-powered agricultural decision-support platform designed to bridge the gap between traditional farming and modern technology. It empowers farmers to detect crop diseases, predict yields, analyze weather patterns, and discover financial aid through an incredibly intuitive, premium dashboard.

The platform provides a **complete digital ecosystem** for modern agriculture, integrating:

- рҹ”¬ Real-time crop disease detection via **Google Gemini Vision AI**
- рҹӨ– Conversational AI farming advisor powered by **Gemini LLM**
- рҹҢҰ Agricultural-specific weather forecasting and disaster alerts
- рҹ“Ҳ Data-driven crop yield prediction and profitability analysis
- рҹҸӣ Smart matching for government agricultural schemes and subsidies

```
рҹ“ё Upload a leaf image  вҶ’  рҹ§  AI detects the disease  вҶ’  рҹ’Ҡ Get treatment in seconds
```

> No expertise required. No complicated setup. Just point, scan, and grow.

### рҹҢҝ AI Crop Disease Detection
> Upload any crop leaf image вҖ” get a precise diagnosis instantly.
- **Drag-and-drop** or camera capture on any device
- **Gemini Vision AI** evaluates the crop's health dynamically
- Returns: disease name, confidence score, symptoms, treatment plan, and prevention tips

### рҹӨ– AI Agricultural Advisor
> Your personal farming expert вҖ” available 24/7.
- Ask anything: disease ID, pesticide selection, fertilizer ratios, irrigation timing
- **Context-aware** responses based on real-world agronomy data
- Clean, chat-like interface powered by **Google Gemini**

### рҹ“Ҳ Yield & Profit Optimizer
> Plant smarter. Earn more. Waste less.
- Calculate expected crop yield based on farm size, soil type, and current climate conditions
- Helps farmers plan logistics, storage, and market sales before the harvest even begins

### рҹҸӣ Government Scheme Explorer
> Never miss a subsidy or support program again.
- Filters national and state-level agricultural schemes based on the farmer's specific profile
- Unlocks hidden funding, subsidies, and insurance policies to protect the farmer's livelihood

## рҹҡҖ Installation & Setup

### Prerequisites
| Tool | Min Version | Download |
|---|:---:|---|
| Node.js | v18+ | [nodejs.org](https://nodejs.org/) |
| Git | Latest | [git-scm.com](https://git-scm.com/) |

### Step 1 вҖ” Clone the Repository
```bash
git clone https://github.com/abhishek-ms-01/agri-intelligence-platform.git
cd agri-intelligence-platform
```

### Step 2 вҖ” Set Up Environment Variables
Create a `.env` file in the **backend** folder:
```env
PORT=5000
MONGODB_URI=mongodb+srv://<user>:<password>@cluster0.xxxxx.mongodb.net/cropcare
GEMINI_API_KEY=AIzaSy_your_gemini_api_key_here
NODE_ENV=development
```

### Step 3 вҖ” Run the Application
```bash
# Terminal 1 вҖ” Start backend server
cd backend
npm install
npm run dev

# Terminal 2 вҖ” Start frontend dev server
cd frontend
npm install
npm run dev
```

Open your browser to `http://localhost:5173`
